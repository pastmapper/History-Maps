About This Project

This website presents interactive maps based on historical land records from the Bureau of Land Management (BLM) General Land Office. The maps are designed to help explore how land was distributed to individuals during the early settlement of Skagit County and surrounding areas.

Purpose

The purpose of this project is educational. It provides a visual way to examine patterns of early land claims, settlement, and federal land distribution using modern mapping tools and historical source records.

Data sources

  • Bureau of Land Management (BLM) General Land Office patent records
  • Public Land Survey System (PLSS) cadastral data
  • OpenStreetMap base map layers

Accuracy and limitations

Parcel boundaries shown on these maps are approximate and are intended for historical visualization only. These maps should not be used for legal, surveying, or property boundary determination.

About the author

This project was created by Corrinne Sande in 2026 as part of an effort to document and visualize early federal land claims and settlement history in Skagit County, Washington.
